# HNG14-STAGE0-TODO-Item-Card
A clean, responsive, and accessible Todo Card component built with **HTML, CSS, and JavaScript**, designed to meet strict testing and UI requirements.

## 🚀 Live Demo
🔗 https://your-live-link-here

## 📌 Features

- ✅ Fully testable with required `data-testid` attributes  
- ♿ Accessible (ARIA labels, semantic HTML, keyboard support)  
- 📱 Responsive design (mobile → desktop)  
- ⏱ Dynamic time-remaining calculation  
- 🔄 Real-time updates (every 30 seconds)  
- ✔️ Interactive completion toggle (updates UI + status)  
- 🏷 Tag system with flexible layout  
- ✏️ Edit & 🗑 Delete actions (simulated interactions)

## 🧱 Tech Stack

- HTML5 (Semantic structure)
- CSS3 (Modern UI, Flexbox, responsive design)
- Vanilla JavaScript (DOM manipulation & logic)

## 🧠 Key Implementation Details

- Used `<article>`, `<time>`, `<button>`, and `<ul>` for semantic correctness  
- Implemented dynamic time calculation using JavaScript Date API  
- Ensured accessibility with `aria-label`, `aria-live`, and proper form controls  
- Designed for testability with exact `data-testid` attributes
